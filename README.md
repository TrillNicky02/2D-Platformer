# 2D-Platformer
Project 4 Part2 for MSCH-C220

Oct 24, 2023

Description
Part2 of a Platformer game. To play use the Keyboard.

## Implementation
* Built using Godot 4.1.1
* A State Machine is implemented
* 2 levels are implemented
* Tile Map nodes are implemented
